# VC Executive Dashboard 🚀

Welcome to your AI-powered business dashboard! This application is designed to give you instant insights into your sales performance using Gemini AI.

The dashboard reads data from a local CSV file by default, but it is **fully ready** to connect to Supabase for a real-time database experience.

---

### 🔌 How to Connect Supabase (Optional)

If you want to move from a CSV file to a real database, follow these simple steps:

1.  **Create a Supabase Project**: Go to [Supabase](https://supabase.com/) and create a new project.
2.  **Create the Table**: 
    - In your Supabase dashboard, go to the **SQL Editor** (the `>_` icon).
    - Paste and run this script to create your data table:
    ```sql
    CREATE TABLE sales_data (
      id BIGINT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
      date DATE NOT NULL,
      product TEXT NOT NULL,
      channel TEXT NOT NULL,
      revenue DECIMAL(15, 2) DEFAULT 0,
      orders INTEGER DEFAULT 0,
      cost DECIMAL(15, 2) DEFAULT 0,
      visitors INTEGER DEFAULT 0,
      customers INTEGER DEFAULT 0,
      created_at TIMESTAMPTZ DEFAULT NOW()
    );

    -- This allows the dashboard to read your data securely
    ALTER TABLE sales_data ENABLE ROW LEVEL SECURITY;
    CREATE POLICY "Public Read Access" ON sales_data FOR SELECT USING (true);
    ```
3.  **Import Your Data**: 
    - Go to the **Table Editor** in Supabase and select the `sales_data` table.
    - Click **Insert** -> **Import Data from CSV** and upload your sales file.
4.  **Get Your Credentials**:
    - Go to **Project Settings** -> **API**.
    - Copy your `Project URL` and `service_role` (or `anon`) key.
5.  **Configure Your Local App**:
    - Open the `.env` file in this folder.
    - Update your credentials and set the `VITE_DATA_SOURCE` to `supabase`:
    ```env
    VITE_DATA_SOURCE=supabase
    VITE_SUPABASE_URL=your_project_url
    VITE_SUPABASE_ANON_KEY=your_anon_key
    ```
6.  **Restart & Enjoy**:
    - Restart your development server (`npm run dev`).
    - The dashboard will now serve data directly from Supabase!

### 🧠 Gemini AI Insights
The "AI Consultant" panel uses Google's Gemini models to analyze your metrics. 
Simply ensure your `VITE_GEMINI_API_KEY` is set in the `.env` file to generate alerts and suggestions.

- **Charts**: Recharts
- **Database Ready**: Supabase
- **AI Engine**: Google Gemini
