# VC Executive Dashboard 🚀

Welcome to your AI-powered business dashboard! This application provides instant insights into your sales performance using Gemini AI and Supabase.

## 📦 Deployment on Vercel

This project is optimized for [Vercel](https://vercel.com/) using the Next.js App Router.

1.  **Import to Vercel**: Connect your GitHub repository.
2.  **Environment Variables**: Add these in Vercel Project Settings:
    - `NEXT_PUBLIC_SUPABASE_URL`: Your Supabase Project URL.
    - `NEXT_PUBLIC_SUPABASE_ANON_KEY`: Your Supabase Anon Key.
    - `NEXT_PUBLIC_GEMINI_API_KEY`: Your Google Gemini API Key.
3.  **Deploy**: Click deploy and your dashboard is live!

---

## 🛠️ Local Setup

1.  **Install Dependencies**:
    ```bash
    npm install
    ```
2.  **Run Development Server**:
    ```bash
    npm run dev
    ```
3.  **Access App**: Open [http://localhost:3000](http://localhost:3000).

---

## 🔌 Supabase Database Setup

1.  **Create Project**: Create a new project on [Supabase](https://supabase.com/).
2.  **SQL Schema**: Run this in the Supabase SQL Editor:
    ```sql
    CREATE TABLE sales_data (
      id BIGINT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
      date DATE NOT NULL,
      product TEXT NOT NULL,
      channel TEXT NOT NULL,
      revenue DECIMAL(15, 2) DEFAULT 0,
      orders INTEGER DEFAULT 0,
      cost DECIMAL(15, 2) DEFAULT 0,
      visitors INTEGER DEFAULT 0,
      customers INTEGER DEFAULT 0,
      created_at TIMESTAMPTZ DEFAULT NOW()
    );

    ALTER TABLE sales_data ENABLE ROW LEVEL SECURITY;
    CREATE POLICY "Public Read Access" ON sales_data FOR SELECT USING (true);
    ```
3.  **Import Data**: Use the Table Editor to upload your CSV data to the `sales_data` table.

## 🧠 Gemini AI
The dashboard uses Google's **Gemini 2.0 Flash** to provide business strategies and alerts based on your sales data.
